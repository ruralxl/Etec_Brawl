Bem vindo ao Etec-Brawl


Neste website iremos te mostrar como foi o processo de criação do nosso jogo "Etec-Brawl". Este jogo foi originado de um trabalho escolar cujo objetivo era usar o gamemaker para fazer um jogo, assim tivemos a ideia de fazer um jogo de luta entre alguns de nossos professores. Veja a seguir o processo de confecção do jogo

Processo de Programação


Para a programação, usamos o sistema de objetos para representar tudo que possa ser interagivel no jogo. Nas imagens abaixo vamos explicar como foi o processo de programação do jogo no Game Maker.


Create e vida

Aqui temos um "create", ou seja, variaveis que são criadas assim que o objeto for iniciado. Neste create, diz que quando o objeto for criado ele tera uma variavel chamada "lifeb", que será a vida do nosso personagem.


Movimentação

Este é uma parte do código de movimentação para o lado esquerdo. Onde ele movimenta a imagem para a esquerda ja que em "x" o valor é -=2. No comando "sprite_index = spj_andando" faz que ao clicar na tecla, alem de movimentar o personagem ira também trocar o sprite. E por fim, o comando "image_xscale = -1" serve para dimensionar o x do objeto no layout do jogo.


No exemplo abaixo temos o mesmo processo, mas agora para o lado direito.


Soco

Este é o código de soco, que diz que ao apertar a tecla de soco, tal ato é feito, Quando o soco é dado, ele ativa um código que diz que caso o objeto hitbox não exita (oque sera explicado mais para frente), ele o cria.


Step e animação

Aqui vemos um "step" que indica algumas ações, este step diz que quando tal sprite existir, ela sera mudado na mesma hora para oque esta dentro das chaves, oque não afetara a movimentação, ja que este código so afeta quando a ação parar. Este código serve para que a animação não continue após o fim da ação.


Hitbox

Aqui temos a hitbox, o objeto que vai contar o dano e retirar a vida. Ela detecta colisão com o jogador oponente, caso a colisão aconteça, ela sera colocada em uma lista em forma de variavel, e essa variavel sera utilizada para retirar vida do personagem.


Jogar e End

Os códigos a seguir servem para iniciar e terminar o jogo, eles são objetos que ao serem clicados, fazem suas ações.
